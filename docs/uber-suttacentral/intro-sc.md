# SuttaCentral
[SuttaCentral](https://suttacentral.net) ist eine internationale Webseite mit frühen buddhistischen Texten in ihren Originalsprachen und zahlreichen Übersetzungen sowie vielen Parallelen. In diesen einführenden Texten können Sie mehr über die Idee und den Aufbau von SuttaCentral erfahren.
